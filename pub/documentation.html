API Documentation

Getting Started

First, need to include these tags in html file.

< link rel="stylesheet" type="text/css" href="https://history-library.herokuapp.com/History.css">
< script type="text/javascript" src="https://history-library.herokuapp.com/History.js">

Creating History Object

The History object takes in few arguments, which consists of startYear: An Integer which is the starting year values for the history object; endYear: An Integer which is the ending year values for the history object; width: An Integer which is the width of the whole history object element; height: An Integer which is the height of the whole history object element; selector: A class or id selector that the user want to append to; direction: A string which takes in values of either "horizontal" or "vertical". Indicating which direction that the user want the History object. side: A string which takes in values of either "both", "up", "down", "left", "right". Note that if choesn direction as "horizontal", the side can only takes in "up", "down", or "both". If chosen "vertical", the side can only takes in "left", "right", or "both". The side represents which side the element should be, either left side of the svg object if chosen "left" or right side if the svg if chosen "right".

const h = new History(startYear, endYear, width, height, selector, direction, side)
const h = new History(2010, 2023, 400, 500, '.featureOneExampleLeft',"horizontal", 'both')

Then all the methods can be call via the object

h.methodName(...)

Methods Documentation

1: createNonAnimatedClickHistory()

Method Code:
createNonAnimatedClickHistory(includeYear, events, barLength)
Description:

createNonAnimatedClickHistory takes in three arguments, includeYear, events, and barLength. The includeYear is a boolean value, indicating whether the user want to show the start year and end year on their History line. The input events is a list of events, in which every object in the list takes the for of {year: ..., name: ..., description: ...}. The last input barLength indicate that how long the user want the bar that grow out of main line. Example is provided in Examples page. The method will create a history time line that each time event is click to show, and the bar growing from the main line is not animated.

Return Values:

createNonAnimatedClickHistory returns a svg element which contains the nessary components to create the History such as Line, Circle, ForeignObject, Text, Button.

5: createScroll()

Method Code:
createScroll(numSteps, idList)
Description:

createScroll takes in two arguments, numSteps, and idList. The numSteps is an Integer values that represents the number of step point that the user want on their scroll bar. The idList is a list that its length is the same as the numSteps and it contains the id that each step point will stop at. (Note that the idList needs to be in the order that the user want to stop. For example, if user want the first stop to be id #first, then the first element in the idList should be #first). Examples is show in the examples page.

Return Values:

createScroll returns a svg element which contains circles that equal to the numSteps.

6: createStep()

Method Code:
createStep(numSteps, htmlList)
Description:

createStep takes in two arguments, numSteps, and htmlList. The numSteps represents the number of the step that the user want the form to have. The htmlList is a list with each element represents a HTML collections. (Note that the list needs to have the order that the user want. For example, by clicking the first step, it will show up the first HTML collections in the first index of the htmlList). Example is provided in the examples page.

Return Values:

createStep returns a svg element which contains circles that equal to the numSteps. And numSteps of ForeignObject element, in each element it will show up the corresponding htmlList's HTML collections.
